Figma Integration

เชื่อมต่อ Figma Design System เข้ากับโค้ดผ่าน VS Code และ Code Connect

Open Figma
Figma File Key 1oi1KLBMz6qVDhAOn7kQOf
  1. 1
    ติดตั้ง Figma VS Code Extension

    ติดตั้ง extension เพื่อดู design spec และ inspect components ได้โดยตรงใน VS Code

    Shell 
    ext install figma.figma-vscode-extension
  2. 2
    ติดตั้ง Figma Code Connect CLI

    ใช้เพื่อ link component ในโค้ดเข้ากับ component ใน Figma

    Shell 
    npm install --save-dev @figma/code-connect
  3. 3
    สร้าง Code Connect mapping

    สร้างไฟล์ .figma.tsx สำหรับแต่ละ component

    Button.figma.tsx 
    import figma from "@figma/code-connect";
import { Button } from "./Button";

figma.connect(Button, "YOUR_FIGMA_URL", {
  props: {
    variant: figma.enum("Variant", {
      primary:   "primary",
      secondary: "secondary",
      ghost:     "ghost",
    }),
    label:    figma.string("Label"),
    disabled: figma.boolean("Disabled"),
  },
  example: ({ variant, label, disabled }) => (
    <Button variant={variant} disabled={disabled}>
      {label}
    </Button>
  ),
});
  4. 4
    Publish ไปยัง Figma

    Push mapping ขึ้น Figma เพื่อให้ทีม design เห็น code reference ใน component panel

    Shell 
    npx figma connect publish
